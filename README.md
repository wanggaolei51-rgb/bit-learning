# 🇮🇩 BIT — Bahasa Indonesia Learning Terminal

> 印尼语学习终端 — 专为冶炼工业园区商务场景设计

## 🌐 在线访问

**GitHub Pages 地址：** `https://[你的用户名].github.io/bit-learning/`

> 部署完成后替换上方地址

## 📱 功能特性

- ✅ **离线可用** — 单文件 HTML，无需服务器
- ✅ **点击查词** — 演讲和场景对话中所有词汇可点击查看
- ✅ **实时翻译** — 右下角 🌐 翻译窗口，支持印尼语↔中文↔英语
- ✅ **词缀拆解** — 自动识别 ber-/me-/di-/pe-/ter-/se- 等前缀 + -kan/-i/-an 后缀
- ✅ **语音朗读** — 内置 Web Speech API，点击 🔊 听发音
- ✅ **每日录入** — 输入当天新学单词，自动生成学习报表
- ✅ **艾宾浩斯复习** — 自动计算下次复习时间
- ✅ **移动端适配** — 支持手机/平板学习

## 🚀 本地使用

直接在浏览器中打开 `index.html` 即可，无需安装任何软件。

```bash
# 或用 Python 启动本地服务器
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```
